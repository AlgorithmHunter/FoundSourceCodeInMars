How to get the data using the api

How to get all the data from the api?. Pass the full url on your browser https://localhost:44320/MathFormulaeAPI/Formulae and a json file with all formulaes will be returned.

How to get data by id from the api?. Pass the url on your browser with the formulae id https://localhost:44320/MathFormulaeAPI/Formulae/id and a json file with all formulaes will be returned.

How to get sorted data from the api?. Pass the url on your browser with the formulae with orderby and order paramater https://localhost:44320/MathFormulaeAPI/Formulae/orderBy/desc and a json file with all formulaes will be returned.

How to edit data from the api?. Pass the url https://localhost:44320/MathFormulaeAPI/Formulae/id on postman or your code with formulae id and the message body should be in json format and http method should be PUT and data will be edited.

How to post data using the api?. Pass the url https://localhost:44320/MathFormulaeAPI/Formulae on postman or your code and the message body should be in json format and http method should be POST and data will be updated

How to delete data using the api?. Pass the url https://localhost:44320/MathFormulaeAPI/Formulae/id on postman or your code with id and http method should be DELETE and data will be deleted e.g DELETE /MathFormulaeAPI/Formulae/id http/1.1 HOST: https://localhost:44320
